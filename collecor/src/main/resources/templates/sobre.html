Objetivos


Site Collector provides a cloud-native data platform for ingesting data from on-premises log sources and active directories into Exabeam SaaS cloud with easy onboarding process, monitoring, and management through centralized cloud user interface.
Site Collector collects messages, transfers, persists, and securely uploads event data; and connects to the Exabeam Security Operations Platform. Site Collector is the managed entry point for logs to be routed to other processing tools, such as Advanced Analytics, Data Lake, Incident Responder, and Case Manager in the Exabeam Security Management Platform. Site collectors gather logs from external servers, systems, and data centers.
